About us

Welcome to the golang.cymru homepage! We are an informal, friendly group of Go programming language enthusiasts based in Wales UK.
The group was set up so there could be a Go 1.6 launch party a little closer to home than the nearest which seemed to be London. We didn’t get there but we now have our first welsh go meetup at cardiff.
Get in touch if you are interested in setting up any other meetup groups anywhere else in the country
